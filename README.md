#  Steam 实时监控 

一个轻量级的纯 Python 脚本，用于实时监控 Steam 好友的动态。不仅能记录好友的上下线状态，还能**精准计算真实游玩时长**并抓取**成就解锁记录**。

## ✨ 核心功能
* **实时状态追踪**：监控好友的上线、下线、开始游戏状态。
* **精准时长计算**：通过比对云端总游玩时长，精确计算单次游戏时间（不受脚本启动时间影响）。
* **成就解锁播报**：抓取并翻译好友在游戏中刚刚解锁的成就。
* **微信推送通知**：接入 Server酱，好友动态实时推送到手机。
* **本地 CSV 归档**：自动生成 `steam_log.csv` 数据报表，方便后续做数据分析。
* **无感开机自启**：内置 Windows 注册表修改逻辑，实现开机后台自动运行（可配置开关）。
